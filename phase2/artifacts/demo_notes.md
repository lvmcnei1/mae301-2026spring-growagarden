# Demo artifact notes

- `demo_phoenix_selected/` shows a hot-climate example using tomatoes, basil, beans, and bell peppers.
- `demo_zone5b_spring/` shows a cooler spring example where the engine shifts toward cool-season crops.
- `phoenix_watering_chart.png` visualizes the weekly irrigation targets for the Phoenix sample.
- `benchmark_results.csv` and `benchmark_summary.md` show the improvement over a static zone/sun baseline.
